## A Jupyter Notebook for Matplotlib drawing

This notebook contains some functions for generating matplotlib figures, which aims to provide an easy-to-use interface and to draw in a consistent style.

## How to use?

Open the notebook, edit the `Prepare Data` block to load your data and plot curves as you need. See the `Functions` block for the usage of functions.
